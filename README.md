Название: Студенческая платформа лояности и мерч-маркетплейс

Описание:

Разработка веб-приложения для мотивации студентов через систему геймификации. Платформа позволяет начислять внутренние баллы за академические и внеучебные достижения (олимпиады, активности) и обменивать их на эксклюзивный университетский мерч в закрытом маркетплейсе.

Ключевой функционал:

Для студентов: Личный кабинет с историей начислений/списаний, корзина, избранное, оформление заказов.

Для администраторов: Управление товарами, создание событий/задач для начисления баллов, генерация отчетов.

Безопасность: Поиск студентов по зашифрованным данным (телефон, номер студенческого, фамилия) с соблюдением стандартов защиты персональных данных.

Стек: Java, Spring Boot, Hibernate/JPA, PostgreSQL.
